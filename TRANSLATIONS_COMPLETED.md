## 🎯 الاستخدام

### تفعيل الترجمات
```python
# في Flask app
from flask_babel import _
message = _('Admin Login')  # سيعرض "تسجيل دخول الإدارة" بالعربية
```

### في القوالب
```html
<h1>{{ _('Welcome back') }}</h1>
<button>{{ _('Sign In') }}</button>
<p>{{ _('Please enter a valid email') }}</p>
```

### تبديل اللغة
```html
<a href="{{ url_for('main.set_language', language='ar') }}">
    {{ _('Arabic') }}
</a>
```
